# Zignolz Public Notebooks

Welcome to the public notebook repository for Zignolz, a fundamentals-based investment research project focused on identifying high-quality S&P 500 companies and building transparent, data-driven strategies.

This repository contains educational and exploratory Jupyter notebooks designed to be opened directly in Google Colab or viewed on GitHub. These notebooks support the free content published on the Zignolz Substack and aim to help readers understand the ideas behind the model, the fundamentals we analyze, and the portfolio techniques we use.
